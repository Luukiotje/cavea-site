# Cavea — Exclusieve & Zeldzame Wijnen

Premium wine membership website for [cavea.wine](https://www.cavea.wine)

## Brand
- **Positioning:** Exclusive, FOMO-driven wine access platform
- **Colors:** Black (#080808), Gold (#c4a24e), Warm white (#f0ece4)
- **Fonts:** Playfair Display (serif headings) + Raleway (sans-serif body)
- **Tone:** Premium, sophisticated, member-first

## Structure
- `index.html` — Complete single-page website
- Logo embedded as base64 SVG (white version)
- Images served from Wix CDN (cavea.wine media)

## Sections
1. Hero with video background
2. Producer ticker bar
3. Four pillars (why Cavea)
4. Producer badges
5. Image divider with quote
6. Persona cards (Investeerder, Verzamelaar, Liefhebber)
7. How it works (3 steps)
8. Blog preview cards
9. Signup form (ready for Mailchimp integration)
10. Footer

## Deploy
Drag `index.html` to [Netlify Drop](https://app.netlify.com/drop) or enable GitHub Pages.

## Blog Content Strategy
- Target: Dutch wine collectors, investors, enthusiasts
- SEO focus: zeldzame wijnen, wijn investering, Bourgogne, exclusieve wijn
- Frequency: Daily posts
- Language: Dutch
